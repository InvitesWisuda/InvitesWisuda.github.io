# TODO: Setup Undangan Wisuda dengan CodeIgniter 3 dan Tailwind CSS

## Step 1: Install XAMPP
- Download XAMPP installer untuk Windows.
- Jalankan installer dan install XAMPP di C:\xampp (default).
- Start Apache dan MySQL dari XAMPP Control Panel.

## Step 2: Download dan Setup CodeIgniter 3
- Download CodeIgniter 3.1.13 dari GitHub.
- Extract ke direktori proyek: d:/Lingkungan Rumah/Undangan Wisuda.
- Konfigurasi database jika diperlukan (untuk statis, mungkin tidak).

## Step 3: Buat Halaman Undangan
- Buat controller Welcome.php di application/controllers/.
- Buat view welcome_message.php di application/views/ dengan Tailwind CSS.
- Gunakan placeholder seperti {{nama}}, {{tanggal}}, dll. untuk data.

## Step 4: Jalankan dan Test
- Jalankan server menggunakan XAMPP.
- Akses http://localhost/Undangan Wisuda untuk melihat undangan.
- Berikan posisi placeholder data kepada user.
